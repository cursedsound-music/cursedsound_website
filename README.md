# CURSEDSOUND

Official GitHub Pages website for CURSEDSOUND.

## Live Site

https://www.cursedsound.com/

## Stack

- HTML + CSS + vanilla JavaScript
- Hosted on GitHub Pages with custom domain via CNAME
- WebGL hero animation rendered directly on canvas

## Project Structure

- `index.html`: Main landing page (hero, about, booking)
- `style.css`: Site styling and responsive layout
- `script.js`: Navigation behavior, smooth scrolling, WebGL hero logic
- `legal_notice.html`: Legal notice
- `privacy_policy.html`: Privacy policy
- `CNAME`: GitHub Pages custom domain mapping
